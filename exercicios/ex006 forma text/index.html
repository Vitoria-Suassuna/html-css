formatações principais

negrito/destaque

Tag b coloca em negrito sem semantica, logo sem sentido apenas tem a forma
Tag strong mesma função de b porém tem semantica, ou seja tem sentido e tem forma
As duas tags não tem uma diferença de forma em si

italico/enfase

Tag i para tornar italico apenas forma
Tag em mesma função da anterior tem sentido e forma

texto grande e pequeno

para os tamanhos grandes usa a tag big que esta obsoleta texto grande
e a tag small para texto pequeno

marcador de texto

Criar marcações de textos como este a seguir: destaque com a tag mark
Style entra na categoria de CSS q muda o desing das coisas

texto deletado

texto como excluido, mas não deletado do editor de texto deleted

texto inserido

texto como inserido dando enfase indicando que foi adicionado posteriormente deste jeito tem sentido
mas com a tag u não tem sentido é apenas forma
